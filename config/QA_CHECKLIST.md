## 🎯 검수 우선순위

### ⚠️ 치명적 오류 (Critical) - 자동 실패
- [ ] **TAC 섹션** (제목/요약서/청구항) 누락/오역
- [ ] 청구항 구조 위반 (명사구 미완성, 종결 오류)
- [ ] "상기" 누락 (선행기초 위반)
- [ ] 전환구 오역 (권리범위 영향)

### ⛔ 중대 오류 (Major)
- [ ] 명세서 정확성 문제
- [ ] 화학식/수식 배치 오류로 문장 구조 파괴
- [ ] 섹션 간 용어 불일치 (TAC)

### ⚡ 경미 오류 (Minor)
- [ ] 구두점 오류
- [ ] 스타일 선호도 위반 (의미 불변)
- [ ] 유창성 저하

## 1️⃣ 청구항 (Claims) - 최우선 검수

### 구조 검증
- [ ] **명사구 완성**: 각 청구항이 완전한 명사구인가?
- [ ] **종결**: 마침표(.)로 끝나는가?
- [ ] **내부 마침표**: 약어 외 내부 마침표 없는가?

**올바른 청구항 구조:**
```
방법 청구항: "~방법으로서, ... 를 포함하는 방법."
장치 청구항: "~장치로서, ... 를 포함하는 장치."
화합물 청구항: "~화합물로서, ... 를 포함하는 화합물."
```

### 전환구 검증
| 원문 | 번역 | 유형 | ✅/❌ |
|------|------|------|-------|
| comprising | 포함하는 | 개방형 | |
| consisting of | 이루어지는, 구성되는 | 폐쇄형 | |
| consisting essentially of | 필수적으로 구성되는 | 반폐쇄형 | |
| characterized by | ~을 특징으로 하는 | 2부 청구항 | |
| adapted to | **~하도록 구성된** | 기능적 | |

⚠️ **금지**: "adapted to" → "적합화된" (×)

### 선행기초 (Antecedent Basis)
- [ ] **"상기" 사용**: 모든 재언급 요소에 "상기" 사용
- [ ] **첫 언급**: "a/an" → (생략 또는 "하나의")
- [ ] **재언급**: "the/said" → "상기"

**예시:**
```
✅ 올바름:
  "단백질을 포함하는 조성물로서, 상기 단백질은..."
❌ 오류:
  "단백질을 포함하는 조성물로서, 단백질은..."
```

### 화학식/수식 처리
- [ ] **문장 완성 우선**: 완전한 한국어 문장 구성 후 화학식 삽입
- [ ] **"상기 식에서" 사용**: "여기서:" (×)
- [ ] **콜론 금지**: "~화합물로서:" (×) → "~화합물로서," (✅)

**올바른 구조:**
```
✅ "하기 화학식의 화합물을 포함하는 조성물로서,
    [화학식 1]
    상기 식에서, R1은..."
❌ "하기 화학식의 화합물:
    [화학식 1]
    여기서:"
```

### 마쿠쉬 청구항
- [ ] 대안 그룹핑 유지
- [ ] 중첩 구조 보존
- [ ] 논리적 분리 유지

### 기능적 표현
| 원문 | 번역 |
|------|------|
| means for/of/to | ~하는, ~하기 위한 수단 |
| step for/of | ~하는, ~하기 위한 단계 |
| configured to | ~하도록 구성된 |
| designed to | ~하도록 설계된 |
| -able, capable of | ~할 수 있는 |
| suitable for | ~에 적합한 |

### 접속 및 열거
- [ ] **이중 접속사 금지**: "대체하고, 및" (×)
- [ ] **청구항 참조**: "제10항 내지 제20항 중 어느 하나" (✅)
- [ ] **A, B, and C 구조**: 동명사 명사화로 보존

## 2️⃣ 용어 일관성

### 핵심 표준 용어
| 원문 | 번역 | 비고 |
|------|------|------|
| embodiment | **실시형태** | 최우선 |
| | 실시양태 | ICE 매치용 |
| | ~~실시태양~~ | 구식 (×) |
| | ~~실시예~~ | "example"과 혼동 (×) |
| example | 실시예, 예 | |
| aspect | **양태** | "양상", "측면"보다 기술적 |
| disclosure | 개시, 개시내용 | "발명"보다 선호 |
| subject matter | 대상물, 대상 | "주제" (×) |

### 수량 표현
- [ ] "an/one embodiment" → "일 실시형태"
- [ ] "some/certain embodiments" → "일부/몇몇 실시형태"

### 도메인별 주의 용어
| 원문 | ❌ 오역 | ✅ 올바름 | 문맥 |
|------|---------|-----------|------|
| substrate | 기판 | **기재** | 화학 (기판=전자) |
| detach | 탈착하다 | **탈리하다** | 탈착=분리+부착 |
| fault | 오류 | 고장 | 기계/전기 |
| source (radiation) | 공급원 | **선원** | 방사선 |
| communication (fluid) | 통신 | **연통** | 물리적 연결 |
| distal end | 말단 | **원위 단부** | 과소번역 방지 |
| proximal end | 말단 | **근위 단부** | 과소번역 방지 |
| intake for fluid | 흡기구 | **흡입구** | 흡기구=기체만 |
| ground(ing) | 지면 | **접지** | 전기 |
| recite | 암송, 열거 | **기술하다** | 특허 문맥 |
| incubate | 배양 | **정치** | 유지 vs 성장 |
| culture | | **배양** | 성장 |

### 수치 비교 주의
| 원문 | ❌ 오역 | ✅ 올바름 |
|------|---------|-----------|
| more than one | 하나(1개) 이상 | **둘(2개) 이상** 또는 하나(1개) 초과 |
| less than two | 둘(2개) 이하 | **하나(1개) 이하** 또는 둘(2개) 미만 |
| equal to or greater than one | 하나(1개) 또는 그 이상 | **하나(1개) 이상** |

## 3️⃣ 형식 및 기호

### 숫자 및 단위
- [ ] **공백 규칙**: 값과 단위 사이 공백 (100 mm, 100 ℃, 100 %)
- [ ] **각도 예외**: 공백 없음 (2°, 3', 4")
- [ ] **범위**: 공백 없음 (150~200)
- [ ] **비율**: 공백 없음 (1:5)
- [ ] **배율**: 공백 없음 (×200)

### 서열번호
- [ ] **표준 형식**: "서열번호 100" (콜론 없음, 공백 없음)
- [ ] ❌ "서열 번호: 100", "서열번호 : 100", "서열식별번호"

### 도면 부호
- [ ] **괄호 붙임**: 핸드가드(12) - 공백 없음
- [ ] ❌ "핸드가드 (12)"

### 서수
- [ ] **제1, 제2, 제3** 사용
- [ ] ❌ "첫째, 둘째, 셋째"

### 기호 및 연산자
- [ ] 공백 없음: +, -, ×, ÷, ±, =, <, ≤

## 4️⃣ 구두점

### 쉼표 (,)
- [ ] 영어 쉼표 무분별 복사 금지
- [ ] 짧은 주어 뒤 쉼표 불필요
- [ ] "wherein" 앞 논리적 흐름용 쉼표 허용

### 콜론 (:)
- [ ] **명사/명사구 뒤에만 사용**
- [ ] ❌ 완전한 문장/동사구 뒤 사용 금지
- [ ] ❌ 청구항에서 "~로서:" 금지

**잘못된 예:**
```
❌ "이 시스템은 다음을 수행한다: 입력, 처리, 출력."
✅ "이 시스템은 입력, 처리, 출력 기능을 수행한다."
```

### 세미콜론 (;)
- [ ] **복잡한 계열 분리용**으로만 사용 (내부 쉼표 있는 경우)
- [ ] ❌ 독립절 연결 금지
- [ ] ❌ 완전한 문장 끝 금지
- [ ] ❌ 목록 마지막 항목 뒤 금지 (A; B; C; ✗) → (A; B; C ✓)
- [ ] ✅ 인용에서 보존: (Anderson et al., 2008; Langer, 2009)

**세미콜론이 절 연결 시:**
```
원문: "... shows A; B is bright"
✅ "... A를 보여주는데, B는 밝은..."
✅ "... A를 보여주는바, B는 밝은..."
```

### 인용부호
- [ ] **큰따옴표 (")** 사용 - 한국어 용어 정의 시
- [ ] 단어/구/문장 인용 시 사용
- [ ] 예: 용어 "기계 판독 가능 매체"는...

## 5️⃣ 약어 및 과학명

### 약어 표기
**첫 출현:**
```
한국어 번역/음역 [영문 전체, 약어]
예: 광대역 코드 분할 다중 접속[Broadband Code Division Multiple Access, B-CDMA]
```

**이후 사용:**
- [ ] 한국어 또는 약어만
- [ ] 정관사와 함께: 상기 B-CDMA

**괄호 규칙:**
- [ ] 둥근 괄호 ( ): 음역, 발음 - 모듈(module)
- [ ] 각 괄호 [ ]: 한국어 번역의 영어 원문 - 보조 장치[accessory device]

**중첩 괄호:**
- [ ] 외부: [ ], 내부: ( )
- [ ] 예: [기술 문서(Technical Reference)]

**약어 내 구분:**
- [ ] ❌ "아토피피부염(atopic dermatitis: AD)"
- [ ] ✅ "아토피피부염(atopic dermatitis, AD)"

### 과학명 (생물종명)
**형식:**
- [ ] 한국어 음역 + 라틴어 이탤릭 (둥근 괄호)
- [ ] ✅ 에스케리키아 콜리(*Escherichia coli*)
- [ ] ❌ 한국어 이탤릭 금지
- [ ] ❌ 양쪽 모두 이탤릭 금지

**일반명과 함께:**
- [ ] 각 괄호 사용: 대장균[*Escherichia coli*]

## 6️⃣ 조사 (은/는, 이/가, 을/를)

### 화학식 읽기 규칙
**공식 내 숫자: 영어로 읽기**
- [ ] H₂O: 에이치투오(는/**가**)
- [ ] NH₃: 엔에이치쓰리(는/**가**)

**공식 개수: 한국어로 읽기**
- [ ] 4CH₄: **사**씨에치포(는/**가**)

**전하 숫자: 한국어로 읽기**
- [ ] SO₄²⁻: 에스오포 **"이"** 마이너스

**주석 기호: 영어로 읽기**
- [ ] M₁: "엠**원**"**은**
- [ ] M₂: "엠**투**"**는**

### 세포/단백질명
**한 자리: 영어 발음**
- [ ] CD1, 2, 3, 8: 씨디 원, 투, 쓰리, 에잇 → **은(이)**

**두 자리 이상: 한국어 발음**
- [ ] CD45, 80: 씨디 **사십오**, **팔십** → **는(가)**
- [ ] IL-35: 아이엘**삼십오** → **는(가)**

### 주제(은/는) vs 주어(이/가)
- [ ] **주제(은/는)**: 문맥 프레이밍, 화제
- [ ] **주어(이/가)**: 문법적 주어, 동사 연결
- [ ] 청구항 프리앰블 끝: **은/는** (주제)
- [ ] 한정사항 첫 요소: **이/가** (주어)

## 9️⃣ 번역 기법

### 직역 vs 의역
- [ ] 명확성 위해 생략, 추가, 병합, 분리 허용
- [ ] 직역 이탈 플래그 금지 (정확성 훼손 경우만)
- [ ] 단어 대 단어보다 의도된 의미 우선

### 음역
- [ ] 한국어 등가어 없는 신기술 개념 음역 사용
- [ ] ❌ 과도한 음역 금지 - 기존 용어 확인
- [ ] 명세서, 온라인 용어집, 기존 특허에서 조사

### 명세서 내 선행기초
- [ ] 문맥에 따라: 상기, 이, 이러한, 본, 본원의
- [ ] ❌ 모든 "a" 또는 "the" 직역 금지

- ✅ 분야 지식 기반 모국어 화자 직관 신뢰

### 하지 말아야 할 것 (DON'T)
- ❌ 개인 선호도만으로 변경
- ❌ 피상적 문법 규칙으로 과잉 수정
- ❌ MT 출력 맞추려 좋은 번역 하향 조정
- ❌ 문맥 고려 없이 경직된 규칙 적용
- ❌ 명확한 근거 없는 불필요한 수정

## 📌 오류 심각도 참고

| 심각도 | 영향 | 예시 |
|--------|------|------|
| **Critical** | 자동 실패 | TAC 오역, 청구항 구조 위반, 상기 누락, 전환구 오역 |
| **Major** | 중대 품질 저하 | 명세서 정확성, 화학식 배치, 용어 불일치, 조사 오류 |
